# TRABAJO PARCIAL DE PYTHON - MODULO 1
**Descripción:**
Este trabajo parcial del módulo 1 de python se desarrolla con visual code, libretas de pandas y de streamlit con los conocimientos que se ha ido viendo en las clases.

En este trabajo de aplicativo con lo que se ha aprendido en clases tiene 4 ejercicios que hablan sobre presupuestos, gastos y retorno de esperado. 

El primer ejercicio verifica los presupuesto total y el gasto total realizado

El segundo ejercicio registrada cada uno de los presupuestos y gastos que se han tenido y los categoriza por inversión o gasto, en cada uno de las actividades registradas se señala si el gasto esta dentro del presupuesto o se excedio.

En el tercer ejercicio se calcula el retorno esperado usando funciones pandas y data frame, para que nos muestre cada una de las actividades registradas en un cuadro con su retorno esperado.

Tenemos tres documentos importantes: app.py, requirements.txt y el README.md
